# Robot Management System??

## roject Overview
This project is a **Robot Management System** developed using **C++ and Object-Oriented Programming (OOP)** concepts.

It simulates a system that manages different types of robots such as:
- Cleaning Robots
- Security Robots
- Delivery Robots

Each robot has its own tasks, maintenance schedule, and status management.

---

## Features
- Add different types of robots
- Assign tasks based on robot type
- Update maintenance schedules
- Charge robots and update battery status
- Search for robots by ID
- Emergency shutdown feature
- Delete robots from system
- Display all robots

---
